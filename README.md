# Course Manager
by Roman Duda

Dived in the world of self-improvement? Started to take different courses, but always wanted to keep track on a budged?

This app should help you:

  * Saving enrolled courses with it's price
  * Editing courses
  * Deleting courses

Tools:
   
   * MVVM
   * Room database
   * Data Binding
   * Intents
   * Adapters
